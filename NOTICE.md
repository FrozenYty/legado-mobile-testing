# Compliance Notice

**Effective Date**: 2026-05-28

This repository previously contained test code and documentation for the Legado Android application.
On learning that the app-under-test was involved in copyright infringement claims, the following
actions were taken immediately:

1. All application source code under `app-under-test/` has been permanently removed.
2. All test code (Espresso, UIAutomator, Unit, Integration) targeting Legado has been removed.
3. All Legado-specific bug reports, screenshots, and test results have been removed.

## What Remains

The project documentation, CI configuration, testing methodology guides, and collaboration
workflows are preserved for educational reference in the mobile testing coursework. These
artifacts are original work by the project team and are not derived from the app-under-test.

## Disclaimer

This repository no longer contains, distributes, or facilitates access to any third-party
copyrighted application code. The preserved documentation and CI infrastructure are generic
mobile testing assets that are application-agnostic.
